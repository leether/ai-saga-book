# 智能涌动

**从 ChatGPT 到智能体时代——时代中的个体如何进入 AI 狂飙**

> 新褶 著 / AI Agent 协作

---

《智能涌动》是一本非虚构纪实作品，记录了从 2022 年 11 月 ChatGPT 上线到 2026 年 AI 智能体时代开启的全过程。全书 34 章，分六卷，覆盖 OpenAI、Google、Meta、Anthropic、微软、苹果、中国大模型公司以及无数普通人在 AI 浪潮中的选择与变化。
